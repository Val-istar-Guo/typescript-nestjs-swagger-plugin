## Usage

<!-- usage -->
Add this plugin to `tsconfig.json`.

```javascript
{
  "compilerOptions": {
    // ...other options
    "plugins": [
      { "transform": "./build/typescript-nestjs-swagger.ts" },
    ]
  },
}
```
